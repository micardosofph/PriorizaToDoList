## Tecnologias

- HTML
- CSS
- JavaScript
- LocalStorage
- Manipulação de DOM

## Funcionalidades

- Landing pages pra apresentar nosso projeto e nossa equipe
- Alternância entre modo claro/modo escuro
- Criar/Excluir, definir uma descrição e marcar como importante uma tarefa
- Busca das tarefas por nome
- Criar e definir categorias para as tarefas
- Calendário para visualizar tarefas por dias


2. Abra index.html no navegador
